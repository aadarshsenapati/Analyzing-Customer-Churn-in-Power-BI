# Analyzing International Call Service

## Overview
This repository contains a Power BI report file, **Analyzing International Call Service.pbix**, designed to provide insights into the performance and customer behavior of an international call service provider. The report encompasses a variety of metrics and analyses to support data-driven decision-making.

## Key Features
The Power BI report includes the following dashboards and insights:

### 1. Data Check
A comprehensive review of the dataset's integrity, including:
- Missing values
- Outliers
- Data consistency

### 2. Churn Rate
A detailed analysis of customer churn, highlighting:
- Overall churn percentage
- Trends over time

### 3. Churn Reason
Insights into the primary reasons for customer churn, providing actionable information to mitigate churn.

### 4. Churn Category
Categorized data on churn, such as voluntary vs. involuntary churn, to better understand customer behavior.

### 5. Churn by State
State-wise breakdown of churn rates, helping identify geographic areas with higher churn.

### 6. Demographics
Analysis of customer demographics, including:
- Age
- Gender
- Income level

### 7. Age Bins
Segmented analysis based on age groups, offering a clear view of churn patterns across different age ranges.

### 8. Inspecting Groups
Comparison of customer groups based on common attributes to uncover patterns and correlations.

### 9. Multiple Fields
Analysis across multiple fields, allowing a multidimensional view of customer behavior and service performance.

### 10. Unlimited Plans
Investigation of customer usage and retention for those with unlimited call plans.

### 11. International Calls
Performance analysis of international call services, including:
- Usage trends
- Customer satisfaction

## How to Use
1. Clone this repository or download the `.pbix` file directly.
2. Open the file in Power BI Desktop.
3. Refresh the data source (if applicable) to connect to updated datasets.
4. Navigate through the dashboards to explore insights.

## Dataset
The dataset used for this analysis contains anonymized customer information and service usage metrics. It includes fields such as:
- Customer ID
- State
- Call Usage
- Plan Type
- Churn Status
- Age

## Objectives
This analysis aims to:
- Identify key factors influencing customer churn.
- Highlight areas for service improvement.
- Support strategic decision-making through data visualization.
